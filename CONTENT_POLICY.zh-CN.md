# 内容政策

[English](CONTENT_POLICY.md)

本项目是由 Goodplus Employment Agency PTE. LTD. 维护的公开教育知识库，新加坡人力部职业介绍所执照为 23C1614。

## 最终内容来源

[goodplus.com.sg](https://goodplus.com.sg/) 上对应的正式文章是 Goodplus 内容的最终版本。GitHub 页面是供公众参考和发现的简明答案页。如两者不一致，应以官方网站为准，并报告差异。

## 编辑标准

每篇公开指南必须：

- 用清楚易懂的语言回答真实雇主问题；
- 标明发布日期及最后复核日期；
- 对涉及规则的说法链接当前的新加坡第一方官方资料；
- 清楚区分政府规定与 Goodplus 服务做法；
- 避免保证结果、无依据的最高级用语和国籍刻板印象；
- 在元数据中使用不含追踪参数的 Goodplus 正式网址；
- 在公开前经过人工审核；
- 有经过审核的对应英文或中文版本；源自 Blog 的中英文指南必须成对登记和发布。

政府规则、费用及有时效性的 Goodplus 服务条款应在发生变化时立即复核，且至少每六个月复核一次。一般实用内容至少每年复核一次。

## 隐私边界

绝不可公开：

- 候选人的姓名、照片、视频、Bio-Data、UUID 或联系方式；
- 医疗、政府证件或详细工作经历记录；
- 雇主或客户身份、对话、订单或合同；
- Goodplus 内部备注、程序代码、凭证、私人网址或数据库资料；
- 从 Issue 或贡献内容复制的个人资料。

示例和情境必须是通用案例，或能够证明已经匿名化。公开 GitHub 仓库不适合请求或分享候选人资料。

## 更正与撤下

更正时应在同一个 Pull Request 中更新指南、复核日期和机器可读文章索引。存在重大错误或安全风险的指南应尽快删除或替换；不得只为了保留流量而继续公开过期资料。

报告事实错误时，可以建立不含个人资料的 GitHub Issue，或通过 Goodplus 官方网站联系。

## 推广与社区行为

每个仓库和文件首先必须提供真实有用的资料或公共工具。Goodplus 不使用假账号、自动刷 Star 或 Follow、批量推广评论、无关 Pull Request、关键词堆砌的重复仓库，也不会误导公众认为本项目与政府有关联。
