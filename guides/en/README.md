# English guides

This directory contains concise English (`en-SG`) answer pages derived from approved Goodplus guides.

Each guide must follow [the answer-pack template](../../docs/GUIDE_TEMPLATE.md), link to its official `goodplus.com.sg/blog/` article and include current first-party sources for rule-sensitive claims.

The first guide batch is being prepared. Use the root [README](../../README.md) for the planned topic list.
